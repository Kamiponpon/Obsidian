
la complejidad logaritmica es una forma de *medir cuanto tarda o cuanta memoria consume un algoritmo*

hay distintos tipos 
$O(n)$ -> Peor caso osea el tiempo maximo posible

$\Omega(n)$ -> Mejor caso osea el tiempo minimo posible

$\Theta(n)$ -> Caso promedio osea tipico tiempo
