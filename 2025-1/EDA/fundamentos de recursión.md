¿que es la recursión?

esto:
```
tipo funcion(tipo parametro){
	if(condicion de parada){
		return algo;
	} else {
		return funcion(caso más simple);
	}
}
```
notar que dentro de la función la llamamos denuevo *eso* es recursividad
