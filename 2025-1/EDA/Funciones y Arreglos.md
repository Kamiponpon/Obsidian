
todos los programas tienen un "main()", esta es la primera función que se llama en el programa siempre
para ocupar las funciones base debemos llamar a 
``#include <stdio.h>``
## Ejemplo de función:
```
int main (){
	int edad = 32;
	printf("tengo %d años \n", edad)
	getchar();
	//comentario aqui
	return 0
}
```
## definiendo una funcion
una funcion se define de esta forma
tipo_de_retorno nombre_de_funcion (lista_de_parametros);
Aqui un ejemplo:
	

```
int max(int num1, int num 2){}
```
*en orden, esta funcion retorna un int, se llama "max" y usa 2 valores numericas int*
